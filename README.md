# Waste Container Map (Expo Web App)

This is a web-only Expo app that displays a map of waste containers (paper and glass). Users can filter markers by waste type using checkboxes.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
